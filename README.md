# Dune

A game based on Frank Herbert's Dune


Dependencies:

    cmake >= 3.16
    make >= 4.2.1
    SDL2 >= 2.0
    gcc/g++ >= 9.3.0
        Windows: recommend using MinGW 


Build Instructions:

    Clone this repo.
    Make a build directory in the top level directory: mkdir build && cd build
    Compile: cmake .. && make
    Run it: ./DuneGame.
